# InstagramClone
A clone of the iOS 10?? Instagram app using the Firebase framework

## Highlights

* Custom tab bar
* UIRefreshControl
* Firebase 5
* AVFoundation
* Swift 4.1
  * List
  * Things
  * Image caching
  * UISearchBar + searching for users
* Custom photo picker using CollectionView
* Photos SDK
